Test for Highcharts employees graph
(URL: http://www.highcharts.com/component/content/article/2-news/146-highcharts-5th-anniversary)

To run this test:
1. Download the contents of repository
2. Open Highcharts.sln file using Visual Studio
3. Build solution
4. Within Visual Studio, open Test Explorer
5. Find TestGreenGraph test
6. Run TestGreenGraph
7. After test has run, you should be able to see the log.txt file containing following information:
date of executing, messages read from the tooltip tested, message about result of the test (passed/failed)
8. Now you cat close project in Studio.
